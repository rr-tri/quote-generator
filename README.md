# Quote Generator App

This is a simple web application built with React that generates random quotes and allows users to tweet them. The app features dynamic color theming and a responsive design.

![Quote Generator Screenshot](Screenshot.png)

## Features

- Display a random quote with its author.
- Generate new random quotes with the "New Quote" button.
- Tweet the current quote with the "Tweet Quote" button.
- Dynamic theming: Background color, text color, and button color change with each new quote.

## Technologies Used

- React: A JavaScript library for building user interfaces.
- Axios: A promise-based HTTP client for making API requests.
- CSS: Used for styling the components.
- [quotable.io](https://quotable.io/): An external API used to fetch random quotes.

## Getting Started

To run this app locally, follow these steps:

1. Clone this repository to your local machine using the following command:

   ```
   git clone https://github.com/your-username/quote-generator-app.git
   ```

2. Navigate to the project directory:

   ```
   cd quote-generator-app
   ```

3. Install the dependencies:

   ```
   npm install
   ```

4. Start the development server:

   ```
   npm start
   ```

5. Open your web browser and visit `http://localhost:3000` to view the app.

## Usage

- Click the "New Quote" button to generate a new random quote.
- Click the "Tweet Quote" button to share the current quote on Twitter.
- Enjoy inspirational and thought-provoking quotes!
